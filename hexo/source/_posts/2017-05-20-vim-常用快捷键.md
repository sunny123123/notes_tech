## 移动
+ h 左移
+ l 右移
+ j 下移
+ k 上移
+ w 移动一个单词
+ W 移动下一个单词开头，但忽略一些标点
+ e 前移一个单词，光标停在下一个单词末尾
+ E 移动到下一个单词末尾，如果词尾有标点，则移动到标点
+ 0 移动到行首
+ $ 移动到行尾
+ gg 移动最前面
+ shift+g 移动到最后面
## 翻屏
+ crtl+f 下翻一屏
+ crtl+b 上翻一屏
+ zz 移动到屏幕最中央
## 复制、粘贴、删除
+ y 复制一行
+ :m,ny 复制m到n行
+ d 删除一行
+ :m,nd 剪切m到n行
+ p 粘贴
## 打开多个文件
+ :sp file  水平打开多个文件
+ :vsp file 垂直打开多个文件
+ ctrl+w 双击快速切换窗口
+ ctrl+w+j/k 快速上下切换
+ u 撤销
+ ctrl+r 反向撤销
